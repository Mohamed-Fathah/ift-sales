# IFT ERP — Islamic Foundation Trust
## Publication Sales & Inventory Management System

Built by **Mohamed Fathah** | [fafacreatives.in](https://fafacreatives.in)

### Live Demo
🌐 https://ift-erp-v2-pi.vercel.app

### About
A full Enterprise Resource Planning system built for Islamic Foundation Trust, Chennai.
Manages book sales, inventory, purchases, expenses and accounts.

### Features
- 📦 Books Catalog & Stock Management
- 🧾 Billing / POS with PDF Receipt & WhatsApp sharing
- 🛒 Purchase Management with transport & unloading charges
- 👥 Customers & Suppliers Management
- 💰 Expense Tracking
- 📊 Sales, Stock & Purchase Reports
- 🔍 Audit Trail & Version Control
- 👤 Multi-user with role-based access

### Tech Stack
- Next.js 14 + TypeScript
- Supabase (PostgreSQL)
- Tailwind CSS
- Vercel Hosting
